# 04-CUSTOM

> Your Custom Zone — Define it however works for you.

---

## What is this?

`04-CUSTOM/` is a **reserved slot for your own structure**. Brain OS doesn't prescribe what goes here — you decide.

This is intentional. Every person's workflow is different. Some things don't fit neatly into knowledge, personal ops, or projects. This is your space for those things.

---

## Suggested Uses

Pick one (or make up your own):

| Option | What to put here |
|--------|-----------------|
| **Domain Focus** | Deep-dive on one area (e.g., `04-CUSTOM/ai-research/`, `04-CUSTOM/language-learning/`) |
| **Side Projects** | Lightweight side project notes separate from main `05-PROJECTS/` |
| **Reading / Media** | Book notes, film logs, podcast highlights |
| **Scratch Space** | Temporary working area for ideas in progress |
| **Templates** | Your own custom templates not in the official `templates/` folder |
| **Reference** | Cheat sheets, quick references, lookup tables |
| **Experiments** | Testing new organizational structures before committing |

---

## Rules

1. **No rules** — structure it however makes sense to you
2. AI won't process this area unless you explicitly direct it to
3. If you find yourself always directing AI here, consider formalizing the structure and contributing back to Brain OS

---

## Getting Started

Start with one folder. Name it something meaningful to you.

```
04-CUSTOM/
└── your-thing/        # e.g., ai-research/, side-hustle/, reading/
    └── README.md      # Optional: explain what's in here
```

That's it.
